## 4. 这套项目的关键规则

1. **不要手改 `index.html`**，应始终通过 `build_vis_github.py` 重新生成
2. **构建脚本只用 `build_vis_github.py`**，不要换成旧脚本
3. **手机原图先压缩**，目标建议 150-350KB，最长边 1600px，JPEG 质量 72
4. **单元图命名规则**：`单元号-序号.jpg/png`
5. **楼层图命名规则**：`floor_楼层号.jpg`
6. **GitHub Desktop 必须连到正确目录**，不要误用只有 README 的空克隆目录
